# **App Name**: ServiceFlow Control

## Core Features:

- Service Control: Implement a Service Control screen to manage service requests and their details.
- Service Commission: Develop a Service Commission screen for managing commissions associated with each service.
- Annotations: Incorporate a Notes screen to record and view notes related to each service or commission.
- PDF Export: Enable PDF export functionality for both Service Control and Service Commission screens.
- AI Note Suggestion: Generate suggested notes. Use the LLM as a tool to incorporate helpful facts in the suggestions, using external documents as needed to accomplish the note taking goal.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke a sense of reliability and efficiency.
- Background color: Light gray (#F0F4F8), offering a clean and professional look.
- Accent color: Warm orange (#FFB74D) for highlighting interactive elements and important notifications.
- Body and headline font: 'Inter', a sans-serif font, for a modern, neutral and easily readable text.
- Use minimalist, line-based icons to represent actions and status within the service management screens.
- Employ a clear and intuitive layout with tab-based navigation to switch between Service Control, Service Commission, and Annotations screens.
- Subtle transitions and animations for screen changes and data loading to enhance user experience.
